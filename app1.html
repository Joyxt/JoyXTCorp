🛡️ La Raison d'Être

Dans notre écosystème numérique actuel, même les messageries dites "chiffrées de bout en bout" collectent de précieuses métadonnées. Tadpole Cipher propose une approche radicale : le chiffrement à la source, totalement déconnecté de tout réseau.

C'est un sas de sécurité qui transforme votre texte en une suite de caractères aléatoires avant même qu'ils ne touchent Internet.

⚙️ La Mécanique Cryptographique

Le système repose sur une dérivation de clé symétrique stricte :

  • Code PIN (4 chiffres) : Seed cryptographique rapide.
  • Phrase Maître (3 mots min) : Apporte l'entropie nécessaire.
Algorithme : SHA-256 + PRNG + Chiffrement octet par octet.

🏗️ Choix Techniques

  • Framework : Flutter (Dart) pour une compilation native.
  • Interface : Gestion d'état via Provider, retour haptique.
  • Esthétique : Shaders GLSL (fluide dynamique à 60 FPS).

🚀 Tutoriel Sommaire

Étape 1 : Identification

Entrez votre PIN et votre Phrase Maître. Cliquez sur l'icône pulsante pour accéder au terminal.

Étape 2 : Le Terminal

Collez votre texte, appuyez sur CHIFFRER. Copiez le résultat et envoyez-le via n'importe quel canal (Discord, WhatsApp, etc.).

🛡️ The Mission

In today's digital ecosystem, even "end-to-end encrypted" messengers collect metadata. Tadpole Cipher offers a radical approach: source encryption, totally disconnected from any network.

⚙️ Cryptographic Mechanism

The core relies on a strict symmetric key derivation:

  • PIN Code (4 digits): Fast cryptographic seed.
  • Master Phrase (3 words min): Provides necessary entropy.
Algorithm: SHA-256 + PRNG + Byte-by-byte encryption.

🏗️ Technical Stack

  • Framework: Flutter (Dart) for native performance.
  • UI/UX: Provider state management, haptic feedback.
  • Visuals: GLSL Shaders (dynamic fluid at 60 FPS).

🚀 Quick Tutorial

Step 1: Identification

Enter your PIN and Master Phrase. Click the pulsing icon to enter the terminal.

Step 2: Tactical Terminal

Paste your text, press ENCRYPT. Copy the result and send it via any app.
